## Engenharia de Prompts (Testes e Cicatrizes)
### Perguntas Técnicas
- Como aplicar *few-shot prompting* em modelos de linguagem?  
- Quais são as diferenças entre *chain-of-thought* e *least-to-most prompting*?  
- Como otimizar prompts para reduzir *hallucinations* em LLMs?  
- Quais frameworks podem ser usados para criar prompts consistentes e previsíveis?  

### Perguntas Simples
- O que é um prompt?  
- Por que a clareza é importante na criação de prompts?  
- Como a engenharia de prompts pode ajudar estudantes e profissionais?  
- Qual a diferença entre um prompt simples e um avançado?  

### Dificuldades Encontradas
- Ajustar prompts para evitar respostas vagas.  
- Encontrar equilíbrio entre simplicidade e profundidade.  
- Garantir consistência nas respostas da IA.  
### Exemplo Prático
**Prompt:**
"Explique o que é engenharia de prompts para um iniciante, com exemplo prático."

**Resposta esperada:**
Explicação simples + exemplo real

**Aprendizado:**
Prompts com contexto claro geram respostas mais úteis.

### Iteração de Prompt

**Versão 1:**
"Explique engenharia de prompts"

**Problema:** Resposta muito genérica

**Versão 2:**
"Explique engenharia de prompts para iniciantes com exemplos práticos do dia a dia"

**Resultado:** Resposta mais clara e aplicável

**Aprendizado:** Quanto mais contexto, melhor a resposta

## Miniguia de Estudo
### 📑 Resumos Estruturados
Engenharia de Prompts é a prática de criar instruções claras e eficazes para modelos de linguagem, permitindo extrair respostas mais úteis e consistentes.  

### Glossário
- **Prompt**: Instrução dada à IA.  
- **Few-shot prompting**: Técnica que fornece exemplos para guiar a resposta.  
- **Chain-of-thought**: Método que força a IA a mostrar o raciocínio passo a passo.  
- **Hallucination**: Quando a IA inventa informações sem base real.  
- **Zero-shot**: Prompt sem exemplos  
- **LLM**: Modelo de linguagem de grande escala  
- **Token**: Unidade de texto processada pela IA
